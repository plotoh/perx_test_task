# test_service

core-модуль для fastapi‑приложения, предоставляющий единые интерфейсы для работы с postgresql и redis.

## архитектура

проект разделён на две части:
- **core** — инфраструктурное ядро (подключение к бд, редису, конфиг, логи, ошибки, зависимости).
- **примеры использования** (`api_example`, `models_example`, `repositories_example`, `services_example`) — показывают, как остальные модули могут использовать core.
